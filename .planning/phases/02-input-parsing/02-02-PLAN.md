## Goal

Add audio transcription to GeminiService and create a timestamp correlation system that links spoken component names to their corresponding video frames.

## Tasks

### Task 1: Add transcription method to GeminiService

<files>backend/services/gemini_service.py</files>

<action>
Add `transcribe_audio(audio_bytes: bytes) -> dict` method:
1. Sends audio blob to Gemini 2.0 Flash with `audio_timestamp=True`
2. Returns structured transcription:
```json
{
  "full_text": "The hydraulic cylinder looks good, no leaks...",
  "segments": [
    {
      "text": "The hydraulic cylinder",
      "start_time": 0.5,
      "end_time": 1.8,
      "component_mentioned": "hydraulic cylinder"
    },
    {
      "text": "looks good, no leaks",
      "start_time": 1.8,
      "end_time": 3.2,
      "component_mentioned": null
    }
  ],
  "components_mentioned": [
    {"name": "hydraulic cylinder", "timestamp": 1.0}
  ]
}
```
3. Prompt tells Gemini to use Caterpillar's exact terminology
4. Prompt extracts component names mentioned in speech
</action>

</verify>

### Task 3: Timestamp correlation utility

<files>backend/services/gemini_service.py</files>

<action>
Add `correlate_timestamps(transcription: dict, frame_timestamps: list) -> dict` method:
1. Takes transcription output and frame capture timestamps
2. When audio mentions a component at time T, finds the nearest frame(s) to T
3. Tags those frames as "likely depicting [component name]"
4. Returns correlation map:
```json
{
  "correlations": [
    {
      "component": "hydraulic cylinder",
      "audio_timestamp": 1.0,
      "nearest_frame_index": 2,
      "nearest_frame_timestamp": 1.0,
      "confidence": "high"
    }
  ]
}
```
</action>

<verify>
```bash
python3 -c "
from services.gemini_service import GeminiService
svc = GeminiService()
# Test with mock data
corr = svc.correlate_timestamps(
    {'components_mentioned': [{'name': 'bucket', 'timestamp': 1.5}]},
    [0.0, 0.5, 1.0, 1.5, 2.0]
)
print('Correlation:', corr)
"
```
</verify>

<done>Timestamp correlation correctly maps audio mentions to nearest video frames</done>
