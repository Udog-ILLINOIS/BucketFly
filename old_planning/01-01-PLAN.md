<task type="auto">
  <name>Task 2: Create useMediaCapture hook with key frame extraction</name>
  <files>frontend/src/hooks/useMediaCapture.js</files>
  <action>
Create a custom React hook `useMediaCapture` that wraps the MediaRecorder API:

```js
export function useMediaCapture({ frameInterval = 500 }) {
  // State: 'idle' | 'requesting' | 'recording' | 'processing' | 'done'
  // Returns: { status, startRecording, stopRecording, frames, audioBlob, previewRef, error }
}
```

**Implementation details:**
1. `startRecording()`:
   - Call `navigator.mediaDevices.getUserMedia({ video: { facingMode: 'environment' }, audio: true })`
   - Create a MediaRecorder instance with the stream
   - Set up a hidden `<canvas>` to extract key frames
   - Use `setInterval` at `frameInterval` ms to:
     - Draw current video frame to canvas
     - Call `canvas.toDataURL('image/jpeg', 0.8)` to capture as base64
     - Push to frames array
   - Record audio chunks via MediaRecorder `ondataavailable`
   - Attach stream to `previewRef` (a useRef for the video element)

2. `stopRecording()`:
   - Stop the interval timer
   - Stop MediaRecorder
   - Stop all tracks on the stream
   - Collect audio blob from recorded chunks
   - Set status to 'done'
   - Return { frames: string[], audioBlob: Blob }

3. **Key frame preservation (USER DECISION):** Err on side of keeping MORE frames. Use 500ms interval (2fps). Do NOT downsample or discard frames. All captured frames are kept.

4. Error handling:
   - Camera/mic permission denied → set error message, stay in 'idle'
   - MediaRecorder not supported → set error message

5. Cleanup: Stop all streams and clear intervals on unmount
  </action>
