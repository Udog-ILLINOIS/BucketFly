  <action>
1. Create `frontend/src/services/api.js`:
   ```js
   const API_BASE = import.meta.env.VITE_API_URL || 'http://localhost:5000';

   export async function uploadInspection(frames, audioBlob) {
     const formData = new FormData();
     formData.append('frames', JSON.stringify(frames));
     if (audioBlob) {
       formData.append('audio', audioBlob, 'audio.webm');
     }
     
     const response = await fetch(`${API_BASE}/api/inspect`, {
       method: 'POST',
       body: formData,
     });
     
     if (!response.ok) {
       throw new Error(`Upload failed: ${response.status}`);
     }
     
     return response.json();
   }
   ```

2. Update `CaptureZone.jsx` to call `uploadInspection` after recording stops:
   - In the stop handler: call `uploadInspection(frames, audioBlob)`
   - Show "Uploading X frames..." during upload
   - On success: show "Upload complete! ✓" with frame count
   - On error: show error message with retry option
   - After successful upload, reset to idle state after 3 seconds

3. Add `.env` file to frontend with `VITE_API_URL=http://localhost:5000` (default)
